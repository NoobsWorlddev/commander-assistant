# Commander Assistant

**Commander Assistant – Ghost Edition** is a voice assistant tool designed to enhance the Star Citizen gameplay experience through custom voice commands.

Control your ship with your voice, create custom commands, personalize assistant replies, and follow the project as it evolves toward the official v1.0 release.
